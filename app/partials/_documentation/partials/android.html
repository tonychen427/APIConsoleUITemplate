Install Login Portal for Java

To get started with 8x8 for Java, we recommend you add the SDK to your project using Maven. Install the Java SDK

To install the 8x8 SDK for Java, add the SDK by editing your project's "pom.xml" and adding this to the section.

    
        
            <dependency>
                <groupId>com._8x8.core</groupId>
                <artifactId>dropbox-core-sdk</artifactId>
                <version>2.0.3</version>
            </dependency>

If you are using Gradle, then add this to the dependencies section of your "build.gradle" file.

    
        
            dependencies {
            // ...
            compile 'com._8x8.core:_8x8-core-sdk:2.0.3'
            }

That's it! Now you're ready to get started with the tutorial.
